iCONECT Certification

iCONECT is an award-winning eDiscovery vendor and one of the early industry leaders. With ICONECT XERA, users can organize and analyze documents through a visually rich and intuitive web-based interface.

iCONECT currently offers 4 certification training tracks, all of which can be obtained for free if you have the latest version of the software! If you manage to obtain the three iCONECT foundational certifications then you can call yourself an iCONECT Certified Administrator. The foundational certifications are:

  • Certified Power Reviewer
  • Certified Project Manager Course
  • Certified System Manager
iCONECT also offers specialized certifications. Currently, only the Certified Analytics Reviewer is available with Certified Production Basics and Certified Analytics Administrator coming soon!

ICONECT Certified Power Reviewer

To become an iCONECT Certified Power Reviewer, students must watch all the videos in the Power Reviewer series. The Power Reviewer series has 11 modules comprised of 34 videos.

iCONECT Certified Project Manager

iCONNECT Project Manager classes are designed for litigation support analysts, database and project managers, and others who want to learn how to create databases, load data, and manage their iCONECT servers.

In this course you will learn to:

  • Set up a workflow for a pre-loaded client and project so users can easily review documents and their relationships in the collection
  • Create users, user groups, and user roles for optimal system setup and security
  • Create and manage fields, and create folder and coding layouts that are easy to use and speed up review
  • Create annotation, comment, and batch sets to ensure different review groups have access to the content they need
  • Learn to set up and run iCONECTâ€™s Search Term Report to quickly and easily find key content and folder it without opening the project
  • Explore powerful mass actions.
  • Set up Productions and Quick Productions in iCONECT

iCONECT Certified System Manager

To achieve the iCONECT System Manager certification, students must take the instructor-led System Manager course. The System Manager course is hosted over Zoom, where an iCONECT expert teaches you advanced functions and server tasks for iCONECT.

In this course you will learn to:

  • Create clients, user groups, user roles, and projects, including creation of project structures for optimal review purposes
  • Use project and user group templates to easily set up new client-matter setup with consistent security
  • Data integration: images, natives, OCR; best practices to ensure that rolling loads are optimally integrated; add data easily using iCONECT templates and imports
  • Retain your brand in the interface using branding options
  • Use iVIEW graphs to give team members insight into data, review status, and more
